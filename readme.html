Setup Instructions

Setting up the Webhook

  1. Update the .env file with your WhatsApp Business API values.
  2. Install ngrok
  3. Install the dependecies: npm install
  4. Run the project by: npm run start
  5. ngrok http http://localhost:3040 (Please install ngrok before run it)
  6. Copy the ngrok URL with endpoint, verify token and set it as the webhook URL in your WhatsApp Business API settings.(WhatsApp->Configuration)
    Ex: https://d5c7-103-47-106-241.ngrok-free.app/api/webhook
  7. Click on verify and save button

    8. OR

  8. Watch video tutorial: Webhook Setup Video

Setting up the Flow Webhook

  1. Update the .env file with your WhatsApp Business API values.
  2. Install ngrok
  3. npm install
  4. Run the project by: npm run start
  5. ngrok http http://localhost:3040 (Please install ngrok before run it)
  6. In Meta WhatsApp Flows
    1. Copy the ngrok URL with endpoint in your WhatsApp Manager->Flows->Endpoint in setendpoint URI.
      Ex: https://d5c7-103-47-106-241.ngrok-free.app/api/flowwebhook
    2. Add Phone number(Select no)
  7. Sign public key
    1. node src/util/keyGenerator.js {passphrase}
    2. Open postman collection and fork it.
    3. Go to WhatsApp Flows API collection->Setup Endpoint Encryption->Set Encryption Public Key
    4. Update Authorize token and business phone number id in postman
    5. Copy public key from console and Paste in business_public_key in body in postman
    6. Click on Send, Response should be true
  8. Connect Meta App
    1. Select app(veriy in meta dashboard)
  9. Health Check
    1. Click on Run Health Check
    2. If any error occure check in console and troubleshoot

    OR

  10. Watch video tutorial: Flow Webhook Setup Video

Catalog Setup, Note: It will not work with test number

  1. Go to Commerce Manager, create catalog and add items.
  2. After adding the product go to WhatsApp Manager->Catalog and select the catalog from the list.
  3. Create catalog template with created catalog from Message template
  4. Replace the template details and product id in src/util/interactive.js -> sendSareeProduct
